# VOICE ASSISTANT (TASK 1)

# Description 

This is a simple Voice Assistant developed using Python. 

It listens to the user's voice.

converts speech into text.

performs basic tasks such as speaking responses.

telling the date and time, and opening websites in a browser.

# Features

- 🎤 Speech Recognition
- 🔊 Text-to-Speech
- 📅 Tells current Date and Time
- 🌐 Opens websites in the browser
- 💬 Responds to voice commands

# Project Flow

1.Start
   
2.Initialize Text-to-Speech Engine
  
3.Listen to User Voice

4.Convert Speech to Text

5.Process the Command

6.Perform Action

(Time / Date / Open Website / Respond)

7.Convert Response to Speech

8.Wait for Next Command

9.End


# Technologies Used

- Python
- speech_recognition
- pyttsx3
- datetime
- webbrowser

The assistant listens to voice commands and responds with speech. It can also tell the current date and time and open websites.

# BMI Calculator (TASK 2)

# Description

The BMI Calculator is a simple Python application that calculates a user's Body Mass Index (BMI) using their weight and height. It also categorizes the BMI result into different health categories such as Underweight, Normal Weight, Overweight, or Obese.

# Features

- Accepts user weight (kg) and height (cm)
- Converts height from centimeters to meters
- Calculates BMI using the standard formula
- Rounds the BMI value to two decimal places
- Displays the BMI category
- Easy to use and beginner-friendly

# Technologies Used
- Python

# Project Structure

BMI-Calculator/
│── Santhoshi_TASK2.py
│── README.md


# Project Flow

1.Start
   
2.Enter Weight (kg) & Height (cm)
   
3.Convert Height to Meters
   
4.Calculate BMI
   
5.Display BMI Value
   
6.Determine BMI Category
   
7.Display Health Status
   
8.End


# How to Run

1. Clone this repository.
2. Open the project in VS Code or any Python IDE.
3. Run the program:
   bash
   python Santhoshi_TASK2.py
   
4. Enter your weight and height when prompted.

# BMI Categories

- Below 18.5 – Underweight
- 18.5 to 24.9 – Normal Weight
- 25.0 to 29.9 – Overweight
- 30.0 and Above – Obese
